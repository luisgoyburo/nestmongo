## Description

With MONGODB in localhost server as database

## Installation

```bash
$ npm install
```
npm install mongoose @nestjs/mongoose
npm install --save-dev @types/mongoose



## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
